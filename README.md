# Projeto_VentureGotchi
Projeto de conclusão de curso do Bolsa Futuro Digital, projeto para a empresa VentureGotchi, com boas práticas de trabalho em equipe e a exigência de que cada aluno teste e supervisione o código de outro. A divisão foi feita de forma que todos trabalhem tanto no Backend quanto no Frontend, evitando que apenas um aluno fique com a parte mais pesada.
